# Product App -- Frontend

React application that allows users to:

-   View all products
-   Create a new product (name only)

This frontend communicates with the Spring Boot backend via REST APIs.

---

## Tech Stack

-   React
-   Vite
-   Axios

---

## Running the Application

### 1. Install dependencies

``` bash
npm install
```

### 2. Start development server

``` bash
npm run dev
```

Application will be available at:

http://localhost:5173

---

## Backend Integration

The frontend expects the backend to be running at:

http://localhost:8080

API base URL:

/api/products
